# Update a List with User Input

Wrapper for
[`utils::modifyList()`](https://rdrr.io/r/utils/modifyList.html).

## Usage

``` r
updateList(x, val)
```

## Arguments

- x:

  A list to be updated.

- val:

  Stuff to update `x` with.

## Value

Returns an updated list.
