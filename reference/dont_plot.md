# Suppress Plotting

Suppress Plotting

## Usage

``` r
dont_plot(x, ...)
```

## Arguments

- x:

  Object to call
  [`graphics::plot()`](https://rdrr.io/r/graphics/plot.default.html) on.

## Value

Invisibly returns whatever `plot(x)` would normally returns, but does
not plot anything (which is the point).
